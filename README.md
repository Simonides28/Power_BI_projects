**Power BI Sales Performance Dashboard**

Power BI report analyzing sales performance using a multi-year superstore dataset (2019–2022).
The dashboard focuses on trends, product and category performance, salesperson results, and brand contribution, with an emphasis on clean layout, clear navigation, and business-focused insights.

**Pages**
Executive Overview – high-level KPIs and trends
Product & Category Performance – drill-down analysis of sales, profit, and margins
Salesperson Performance – performance by salesperson and region
Brand Performance – manufacturer contribution and comparison

**Features**
Star-schema data model
Custom Date table and time intelligence (YoY, LY)
Drill-down and drill-through
Consistent navigation and synced slicers

**Data**

Kaggle Superstore dataset (2019–2022)
Salesperson column added for analytical scenarios
